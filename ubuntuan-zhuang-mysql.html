ubuntu安装mysql

Date 周二 21 十月 2014 Tags MySQL，Linux

其实在unbuntu下安装mysql非常简单，只需要执行（需要超级用户的权限）： 

apt-get install mysql-server mysql-client

但是对于ubuntu下，mysql到底安装到哪去了，却找了半天，主要有如下几个目录：

/etc/mysql这个目录下存放了mysql的配置文件，默认是my.cnf。
/var/lib/mysql这个目录存放了mysql的物理存储文件和binlog的日志文件。
/var/log/mysql这个目录存放了mysql的错误日志的文件。
/etc/init.d/这个目录存放了mysql的启动程序，当然也可以通过service的方式启动。
