Windows下使用GITHUB+Pelican搭建个人Blog安装与配置

Date 周五 10 十月 2014 Tags pelican / Python

准备工作

1. 安装Python

到python官网下载Python，最好选择2.7的版本，点击安装，路径如下： 

C:\Python27

将python的安装目录加入到path环境变量中。

2. 安装setuptools

点击这里下载ez_setup.py文件，双击会自动进行setuptools的安装，安装完之后，在C:\Python27\Scripts目录会有一些.exe和.py文件，表明安装完成。
将C:\Python27\Scripts目录加入到path的环境变量中。

3. Pip安装

打开cmd界面，直接输入： 

 easy_install pip

完成安装。

4. Git安装

这个不用多说了，直接到官网去下载，安装就行。

5. Pelican安装

使用已经安装好的Pip来就行安装Pelican，命令如下： 

pip install pelican

6. MarkDown安装

如果你希望你的blog支持markdown的语法，可以使用pip安装，如下： 

pip install markdown

7. Make安装

百度搜索下Make for Windows 下载.exe文件，随便放到一个目录下，并且将该目录添加到path的环境变量中。

Pelican配置

1. 创建项目

在F盘创建blog目录，进入该目录，执行pelican-quickstart命令： 

mkdir blog  
cd blog  
pelican-quickstart

回答一些问题，能按回车就回车带过，否则，随便写一个就行，但是有一点要注意，就是让你回答
这个问题的时候，一定要填写一个网址，随便一个网址都行，不然后面会有各种问题。
命令执行pelican-quickstart完之后，目录结构如下：
其中content是源文件的目录，比如.md文件。 output是生成静态网页的目录。pelicanconf.py是pelican的配置文件。

2.文件撰写

在content目录中，新建first.md文件，编辑如下： 

Title: My super title  
Date: 2010-12-03 10:20  
Category: Python  
Tags: pelican, publishing  
Slug: my-super-post  
Author: Alexis Metaireau  
Summary: Short version for index and feeds  
This is the content of my super blog post.

3.生成静态页面、本地预览

进如blog目录，执行： 

pelican content

生成静态网页。执行： 

make serve

打开本地服务器，这时候，在浏览器中输入： 

http://127.0.0.1:8000

就可以看到效果了。

部署到github上

首先要登录github，然后新建一个仓库，仓库名为username.github.io(username为你github的用户名)，然后执行如下git的命令。 

cd output
git init
git add .
git commit -m "first commit"
git remote add origin https://github.com/xxx/xxx.github.io.git
git push -u origin master

执行完后，在浏览器中输入http://username.github.io就可以看到效果了。
